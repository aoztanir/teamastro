Features

The bot in action!
Productivity
Do many things like browse the web, add and edit tasks, create agendas, set reminders and timers, etc.
Fun
Put memes or add silly overlays for a bit of fun!
Live Data
Get live data of useful things like Corona Virus numbers or time!
Games!
Play games with others on the server like tic tac toe or rock paper sissors!
Play Music
Play any song out there.
Manage
Set moderation rules to block profanity or spam. Astro can also ban/kick people if needed.
